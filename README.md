# 🍕 FoodieHub — Restaurant Ordering Website

A premium dark-themed restaurant ordering web app built with vanilla HTML, CSS, and JavaScript.

## Features

- User registration & login (localStorage)
- Browse 40 menu items (food + cold drinks)
- Product detail page with quantity selector
- Shopping cart with add/remove/update
- Delivery address form with order confirmation
- Order history with cancel & reorder
- About and Help/FAQ pages
- Fully responsive design

## Pages

| File | Description |
|------|-------------|
| `index.html` | Main menu page with auth |
| `product.html` | Product detail & buy now |
| `orders.html` | Order history |
| `about.html` | About the restaurant |
| `help.html` | FAQ & contact form |

## Getting Started

Just open `index.html` in a browser — no build step or server required.

```bash
# Or serve locally with any static server, e.g.:
npx serve .
```

## Tech Stack

- HTML5, CSS3, Vanilla JavaScript
- Google Fonts (Playfair Display, Inter)
- Unsplash images
- localStorage for data persistence
